### PGA2320 preamplifier for AVR microcontrollers
* PGA2320 used as audio signal attenuator
* 6 audio inputs
* Remote control with parser for over 10 protocols such as NEC, RC5, RC5x, RC6, and so on
* Task scheduler
* Simple to learn from
